About this repository

This is an internal knowledge monitoring space for the GenU team. This first prototype was built by MERL team, who scanned key sources in youth livelihoods including ILO, World Bank, UNICEF, Foundations, and others to curate the initial reading list. The interface was prototyped with Claude (Anthropic) and is published via GitHub Pages, currently stored in Hoai's personal GitHub account.

How it works going forward: MERL team scans available sources and collects inputs from GenU colleagues to curate the monthly reading list, then updates the code on GitHub to refresh the page. This is not positioned as a newsletter or a new product. It is a shared reading list of what GenU is tracking.
